Como instalar o seu XPTO? - K19 Eletronics

  1. Verifique se todos os acessórios listados nesta página estão presentes.
  2. Coloque o aparelho na horizontal sobre uma superfície estável.
  3. Conectar o aparelho ao computador ou notebook utilizando o cabo USB.
  4. Usar o CD-ROM para instalação do software.
  5. Conectar o aparelho à fonte de energia com o adaptador AC.
  6. Ligar o aparelho e esperar o reconhecimento no computador.
